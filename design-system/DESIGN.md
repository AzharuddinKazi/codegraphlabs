# CodeGraphLabs

> Category: Brands

> Surface: web

*Structure does the talking. Slate carries every surface, border, and line of text. Crimson exists to mark the one thing that matters.*

A restrained design system using Slate for structure and a single Crimson element per screen for signature accent. Hierarchy comes from weight, not size.

## Color Palette

| Role | Name | Hex | Usage |
| --- | --- | --- | --- |
| background | Base White | `#ffffff` | page canvas |
| surface | Subtle Off-White | `#f7f8fa` | cards and panels |
| foreground | Slate 900 | `#1c2029` | primary text and headings |
| muted | Slate 400 | `#8a90a2` | muted or disabled text |
| border | Slate 100 | `#dfe2ea` | rules and dividers |
| accent | Crimson 600 | `#c81d25` | signature accent, primary CTA, key stat, section divider |
| accent-secondary | Slate 800 | `#2b3040` | secondary actions and emphasis |

## Typography
- **Display:** Manrope — weights 400, 600, 700 — fallbacks: system-ui, -apple-system, sans-serif
- **Body:** Manrope — weights 400, 600, 700 — fallbacks: system-ui, -apple-system, sans-serif
- **Mono:** Source Code Pro — weights 400, 600 — fallbacks: monospace

## Layout

- **Radius:** 8px max, tight scale 2/4/6/8px
- **Border weight:** 1px

### Posture rules
- Nothing rounder than 8px.
- Hierarchy comes from weight (600-700), not dramatic size jumps.
- One crimson element per screen or slide. Three is decoration.
