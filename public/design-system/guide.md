# CodeGraphLabs — Brand Guide

*Structure does the talking. Slate carries every surface, border, and line of text. Crimson exists to mark the one thing that matters.*

A restrained design system using Slate for structure and a single Crimson element per screen for signature accent. Hierarchy comes from weight, not size.

Extracted from designmd://design-md-personal-brand-system.

## Color roles

- **Base White** (`#ffffff`) — background: page canvas
- **Subtle Off-White** (`#f7f8fa`) — surface: cards and panels
- **Slate 900** (`#1c2029`) — foreground: primary text and headings
- **Slate 400** (`#8a90a2`) — muted: muted or disabled text
- **Slate 100** (`#dfe2ea`) — border: rules and dividers
- **Crimson 600** (`#c81d25`) — accent: signature accent, primary CTA, key stat, section divider
- **Slate 800** (`#2b3040`) — accent-secondary: secondary actions and emphasis

## Typography

- Display: Manrope
- Body: Manrope
- Mono: Source Code Pro

## Messaging pillars

- Structure does the talking.
- Scarcity gives value to emphasis.
